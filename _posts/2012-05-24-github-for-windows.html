--- title: Github for Windows layout: post ---

Pengguna windows dimudahkan dengan munculnya github for windows. Fitur yang ada di msysGit seperti Clone Repositories, Commit Changes dan mensinkronkan antara repo lokal dan github hanya dengan satu klik.

Bagi yang bermasalah dengan SSH keys tidak perlu khawatir, github for windows cuma butuh email/username dan password github untuk bisa digunakan.

Cara instalnya cukup mudah, cara menggunakannya juga cukup mudah, lebih lengkapnya langsung ke http://windows.github.com/.
